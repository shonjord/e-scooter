Scootin' Aboot API documentation version 1.0

API endpoints documentation.

This is the API documentation for Scootin' Aboot. Here are some resources that will help you understand the basics of all endpoints.

available scooters.

scooters available to ride.

/clients/scooters/available get

get

returns a collection of available scooters for a mobile client.

get /clients/scooters/available

returns a collection of available scooters for a mobile client.

  • Response

    token to be able to communicate with the endpoint.
